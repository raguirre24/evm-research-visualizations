EVM Implementation Framework

A Practical Guide for New Zealand Construction Projects

5-Phase Implementation Approach

Based on successful case studies and international best practices

1

Foundation Setup

Establish the groundwork for EVM implementation

  • Define project scope and objectives
  • Create Work Breakdown Structure (WBS)
  • Align Cost Breakdown Structure (CBS)
  • Select appropriate tools and systems
  • Train key personnel
2

Baseline Development

Create the performance measurement baseline

  • Develop integrated project schedule
  • Allocate budgets to activities
  • Account for procurement milestones
  • Establish reporting periods
  • Document baseline assumptions
3

System Integration

Connect EVM with existing project systems

  • Link scheduling and cost systems
  • Configure data integration workflows
  • Set up automated reporting
  • Test data accuracy and flow
  • Implement quality checks
4

Performance Tracking

Monitor and measure project performance

  • Update progress regularly
  • Record actual costs accurately
  • Calculate performance indices
  • Analyze variances and trends
  • Generate performance reports
5

Continuous Improvement

Optimize and refine the EVM process

  • Review performance data with teams
  • Identify improvement opportunities
  • Update baselines when necessary
  • Share lessons learned
  • Refine processes and tools
